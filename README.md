# Proiect-de-diploma
Proiectul de licență prezentat are ca scop principal o aplicație web inovatoare care servește ca punte de 
legătură între instituțiile educaționale și părinți, oferindu-le acestora din urmă acces la informații 
actualizate despre activitățile zilnice ale copiilor lor. Aplicația facilitează integrarea unei baze de date 
complexe cu un site ușor de utilizat, unde părinții pot vizualiza detalii despre prezența la cursuri, 
performanțele academice, participarea la activități extracurriculare și alte evenimente relevante.
